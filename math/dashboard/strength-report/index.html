    📈 누적 강점 리포트¶

    생성: 2026-06-08 · 대상: 6개 세트 73문항 (검증 컴파일 62 + 정답풀이 11) 데이터 출처: problems · 약점 지도

    0. 한눈에¶

    • 정답률: 검증한 62문항 전원 정답 (만점 세트: 진도 12/12, 기출C 10/10, 기출A·B 12/12)
    • 확정 약점: 없음. 유일한 주의점은 음수의 제곱근 부호 하나(스스로 정정함).
    • 수준: 미적분Ⅰ(극한·미분) + 공통수학1(이차방정식) 을 고르게, 고난도까지 소화.

    1. 단원별 분포 (개념 연결 기준)¶

    단원 연결 문제 수 평가
    다항함수의 미분 / 미분계수 21 🟢 매우 강함
    함수의 극한 19 🟢 매우 강함
    함수의 연속 13 🟢 강함
    이차방정식 (공통수학1) 13 🟢 강함
    미분가능성 4 🟢 강함(고난도 포함)
    속도와 가속도 3 🟢 양호
    평균값 정리 3 🟢 양호
    그래프와 방정식의 실근 3 🟢 양호

    합이 73을 넘는 것은 한 문제가 두 단원에 걸치기 때문(정상).

    2. 반복 숙달한 핵심 기법 (강점의 실체)¶

    ① 미분계수의 정의로 극한 변환 ⭐⭐⭐¶

    복잡한 극한을 \(\lim\frac{f(x)-f(a)}{x-a}=f'(a)\) 로 꿰뚫어 봄. 근거: C55, C69, C141, 2차 q8, 2차 q9, A159

    ② 다항식 미정계수 결정 ⭐⭐⭐ (최빈·고난도)¶

    "\(f(a)=0 ⇒ (x-a)\) 인수", "두 함수가 공통점/직선 공유 ⇒ 공통 인수", "유한극한 ⇒ 분자가 분모의 0 상쇄". 근거: 진도 q10, 진도 q11, A176, B120, B140, C175, 2차 q11

    ③ 미분가능성·연속 조건으로 함수 확정 ⭐⭐⭐¶

    절댓값 안이 중근(꺾임 없음), 분수꼴 분자가 분모의 0 상쇄, 우함수 ⇒ 도함수는 기함수. 근거: 2차 q12, C119, C175, B83, 단원 q12, C98

    ④ 근과 계수의 관계·대칭식 ⭐⭐⭐¶

    합·곱으로 대칭식 계산, 부호/절댓값 처리, "근 대입으로 차수 낮추기". 근거: 진도 q3, 진도 q6, 진도 q7, 진도 q8, 진도 q12

    ⑤ 극한 계산 기본기 ⭐⭐¶

    \(\infty/\infty\) 최고차 비교, \(0/0\) 유리화, 샌드위치 정리. 근거: 2차 q3~q6, 단원 q4~q6

    ⑥ 그래프·사잇값 추론 ⭐⭐¶

    사잇값 정리 "최악의 경우" 보수적 카운팅, 사차함수 부호 추론, 합성함수 연속. 근거: 1차 q9, 단원 q10, 1차 q7, A151, B116

    3. 유일한 주의점 (보강 1개)¶

    • 음수의 제곱근 부호: \(a<0,\ b<0\) 이면 \(\sqrt a\sqrt b=-\sqrt{ab}\) (그냥 \(\sqrt{ab}\) 아님).
    • 발생: 진도 q4 — 처음 \(-2\sqrt2\) → \(2\sqrt2\) 로 스스로 정정.
    • 처방: 켤레/허근·음수 제곱근이 나오면 부호 한 번 더 확인. (반복 시 약점 등록 예정, 현재는 1회·정정됨)

    4. 종합 평가 & 추천¶

    평가: 미적분Ⅰ(극한·미분·미분가능성)과 이차방정식의 개념 이해 + 고난도 응용이 모두 탄탄. 특히 다항식 결정형과 미분가능 조건형 같은 상위권 변별 문제를 안정적으로 해결.

    추천 다음 단계: 1. 적분(정적분·넓이) 으로 진도 확장 — 미분 기초가 탄탄해 무리 없음. 2. 잠정 9문항 확정 — 그래프/원문만 선명히 받으면 값 확정 (약점 지도 참고). 3. 실전 속도 훈련 — 정확도는 충분하니, 고난도 유형 풀이 시간 단축에 집중. 4. 부호 점검 루틴 — 음수 제곱근/이항 부호만 가볍게 셀프체크.

    다음 시험 대비 시 §2의 ①~④ 기법 문제를 모아 빠르게 복습하면 효율적입니다.
