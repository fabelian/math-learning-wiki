# ⬛ sources/physics/ — 물리 원본 (불변)

물리 과목의 **문제·풀이 원본**을 여기에 넣습니다. LLM은 읽기만 하고 **절대 수정/삭제하지 않습니다.**

---

## 📌 규칙

- 경로: `sources/physics/YYYY/MM/{source-id}.jpg` (문제), `{source-id}-sol.jpg` (풀이), `{source-id}.md` (문제 텍스트, 선택).
- source-id 형식: `YYYY-MM-DD-q{n}` (예: `2026-06-10-q1`).
- 넣은 뒤 [`../index.md`](../index.md) 카탈로그에 한 줄 등록(또는 compile이 자동 등록).
- 개인정보: 사진 속 실명·학교·연락처는 가린다. [`../../docs/privacy.md`](../../docs/privacy.md).

> 첫 물리 원본을 넣고 "compile" 하면 `wiki/physics/`에 문제 페이지가 생성됩니다.
