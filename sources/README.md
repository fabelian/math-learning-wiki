# ⬛ sources/ — 원본 계층 (Raw, 불변)

학생이 넣는 **문제·풀이 원본**이 사는 곳입니다. Karpathy LLM Wiki의 **raw sources** 계층입니다.

> **불변 원칙:** 여기 있는 파일은 진실의 원천입니다.
> LLM은 **읽기만** 하고 절대 수정·삭제하지 않습니다. 위키(`wiki/`)만 갱신됩니다.
> 내용을 바꾸고 싶으면 파일을 고치지 말고 **새 source를 추가**하세요.

---

## 📁 폴더 구조 (연/월)

```text
sources/
├─ index.md            # 원본 카탈로그 (모든 source-id 목록)
└─ YYYY/
   └─ MM/
      ├─ 2026-06-07-q1.jpg       # 문제 사진
      ├─ 2026-06-07-q1-sol.jpg   # 풀이 사진
      └─ 2026-06-07-q1.md        # (선택) 문제 텍스트
```

## 🏷️ source-id 규칙

`YYYY-MM-DD-q{번호}` (예: `2026-06-07-q1`)

| 부분 | 의미 |
|------|------|
| `YYYY-MM-DD` | 푼 날짜 |
| `q1`, `q2` | 그날의 문제 번호 |
| `-sol` | 풀이 사진 (없으면 문제 사진) |

---

## 🔄 사용 흐름

1. 문제/풀이 사진을 `sources/YYYY/MM/`에 규칙대로 저장.
2. (선택) 문제 텍스트를 `{source-id}.md`로 함께 저장하면 분석 정확도 ↑.
3. `index.md`에 한 줄 등록 (또는 compile 시 LLM이 등록).
4. [`prompts/compile.md`](../prompts/compile.md)로 LLM이 위키로 컴파일.

## ⚠️ 주의

- 📦 원본은 가로 1600px 정도로 줄여 저장(용량 절약). 위치정보(EXIF) 제거 권장.
- 🚫 이 폴더의 기존 파일을 수정하지 마세요 — 위키 정합성의 기준점입니다.
