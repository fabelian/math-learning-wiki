# ❌ 오답 분석 (Mistakes)

틀린 문제를 **원인 중심**으로 분석하고, 같은 실수를 반복하지 않도록 교정 전략을 쌓는 공간입니다.

---

## 📌 파일 규칙

- 파일명: `YYYY-MM-DD-주제.md` (예: `2026-06-07-이차방정식-근의공식.md`)
- 새 문서는 [`../templates/mistake-analysis-template.md`](../templates/mistake-analysis-template.md)를 복사해서 만든다.
- 분석한 오답은 반드시 [약점 지도](../dashboard/weakness-map.md)에 반영한다.

---

## 🏷️ 오답 원인 분류 체계 (태그)

분석할 때 아래 태그 중 하나 이상을 붙입니다. 일관된 태그가 통계의 핵심입니다.

| 태그 | 의미 | 예시 |
|------|------|------|
| `#계산실수` | 개념은 알지만 사칙연산·부호·이항에서 틀림 | 부호 바꾸기 누락 |
| `#개념부족` | 정의·공식·정리를 몰라서 틀림 | 근의 공식 조건 모름 |
| `#문제해석오류` | 문제를 잘못 읽거나 조건을 놓침 | "자연수" 조건 무시 |
| `#풀이전략오류` | 접근 방법 자체가 비효율/오류 | 인수분해 가능한데 근의 공식 강행 |
| `#시간부족` | 방법은 알지만 시간 내 못 풂 | 마지막 문제 못 봄 |
| `#실수누락` | 검산·단위·범위 확인 안 함 | 음수 해 버리지 않음 |
| `#서술미흡` | 답은 맞지만 풀이 서술이 부족 | 과정 생략으로 감점 |

> 한 문제에 원인이 여러 개일 수 있습니다. 가장 핵심 원인을 **주원인**으로 표시하세요.

## 🎯 이 영역의 목표

오답은 **부끄러운 기록이 아니라 가장 값진 학습 데이터**입니다.
반복되는 오답 패턴을 찾아내면, 공부 시간을 가장 효율적으로 쓸 수 있습니다.
