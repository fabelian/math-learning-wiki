# 📝 시험 대비 (Exams)

중간·기말고사를 대비하는 공간입니다.
일일 기록과 오답 데이터를 **시험 범위 기준으로 재구성**하여 대비 자료를 만듭니다.

---

## 📌 파일 규칙

- 파일명: `YYYY-학기-시험명.md` (예: `2026-1학기-중간고사.md`)
- 새 문서는 [`../templates/exam-review-template.md`](../templates/exam-review-template.md)를 복사해서 만든다.
- 자료 생성 시 [`prompts/exam-prep.md`](../../prompts/exam-prep.md) 프롬프트를 활용한다.

---

## 🗂️ 시험 목록

| 시험 | 범위 | 대비 상태 | 결과 | 링크 |
|------|------|-----------|------|------|

---

## 🧭 시험 대비 단계

1. **범위 확정** — 교과서/학교 공지로 시험 범위를 적는다.
2. **약점 추출** — [약점 지도](../dashboard/weakness-map.md)에서 범위 내 약한 단원을 뽑는다.
3. **오답 재풀이** — 범위 내 [오답 분석](../mistakes/README.md) 문제를 다시 푼다.
4. **유사 문제 생성** — 약한 유형을 [`generate-similar-problems.md`](../../prompts/generate-similar-problems.md)로 보강한다.
5. **모의 점검** — 시간을 재고 모의 풀이 후 결과를 기록한다.
6. **시험 후 회고** — 실제 결과와 예측을 비교해 다음 시험 전략을 갱신한다.

---

## 🎯 핵심

시험 대비는 "처음부터 다시"가 아니라
**이미 쌓아둔 오답·약점 데이터를 시험 범위로 필터링**하는 작업입니다.
누적된 기록이 많을수록 대비가 빨라집니다.
