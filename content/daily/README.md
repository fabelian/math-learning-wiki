# 📅 일일 기록 (Daily)

매일 푼 문제와 풀이, 그리고 LLM 분석 결과를 **날짜별 문서 한 개**로 쌓는 공간입니다.

---

## 📌 파일 규칙

- 파일명: `YYYY-MM-DD.md` (예: `2026-06-07.md`)
- 새 문서는 [`../templates/daily-problem-template.md`](../templates/daily-problem-template.md)를 복사해서 만든다.
- 하루에 여러 문제를 풀었다면 한 문서 안에 문제별 섹션으로 나눈다.

## 🖼️ 사진 연결

- 문제 사진: `images/YYYY/MM/YYYY-MM-DD-q1.jpg`
- 풀이 사진: `images/YYYY/MM/YYYY-MM-DD-q1-sol.jpg`
- 문서에서는 상대경로로 삽입:
  ```markdown
  ![문제](../../images/2026/06/2026-06-07-q1.jpg)
  ```

---

## 🗂️ 기록 목록 (인덱스)

> 새 일일 기록을 추가하면 아래 표에 한 줄씩 추가하세요. (최신순)

| 날짜 | 단원 | 문제 수 | 맞음/틀림 | 주요 오답 원인 | 링크 |
|------|------|---------|-----------|----------------|------|
| _예시_ | 이차방정식 | 5 | 3 / 2 | 계산 실수 | `2026-06-07.md` |

---

## 💡 작성 팁

- **결과보다 과정**을 적는다. "왜 그렇게 풀었는지"가 핵심.
- 막힌 지점, 헷갈린 순간을 솔직하게 기록한다.
- 틀린 문제는 반드시 [오답 분석](../mistakes/README.md)으로 연결한다.
