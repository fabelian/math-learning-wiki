## 📚 중3 수학 단원 지도

학교 진도/교과서에 맞춰 아래 단원으로 개념 노트를 채워갑니다.

1. 제곱근과 실수
2. 다항식의 곱셈과 인수분해
3. 이차방정식
4. 이차함수
5. 삼각비
6. 원의 성질
7. 통계 (대푯값·산포도·상관관계)
8. 피타고라스 정리

> 각 단원 노트는 [concepts/](concepts/README.md)에서 만들고 관리합니다.

## 🔁 학습 사이클

```text
문제 풀기 → 사진 촬영 → 일일 기록 작성 → LLM 분석
   → 오답 원인 분류 → 약점 지도 갱신 → 유사 문제 풀기 → (반복)
   → 주간 리뷰 → 시험 대비 자료 생성
```

이 사이클이 반복될수록 위키는 **개인 맞춤 학습 데이터**로 성장합니다.
