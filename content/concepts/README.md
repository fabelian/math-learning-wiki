# 📘 개념 노트 (Concepts)

단원별 핵심 개념·공식·정리를 정리하고, **오답에서 발견한 약점을 개념과 연결**하는 공간입니다.

---

## 📌 파일 규칙

- 파일명: `단원명.md` (예: `이차방정식.md`, `이차함수.md`)
- 새 문서는 [`../templates/concept-note-template.md`](../templates/concept-note-template.md)를 복사해서 만든다.
- 오답 분석에서 "개념 부족"이 나오면, 해당 개념 노트로 링크를 건다.

## 💡 작성 원칙

- **공식만 적지 않는다.** "언제 쓰는지", "자주 하는 실수"를 함께 적는다.
- 본인 말로 다시 설명(費曼/Feynman 기법)을 한 줄이라도 남긴다.
- 관련 오답·일일 기록 문서로 양방향 링크를 건다.
