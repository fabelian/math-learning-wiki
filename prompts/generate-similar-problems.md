# 🤖 프롬프트 — 유사 문제 생성 (Generate Similar Problems)

틀린 문제나 약한 유형과 **같은 사고력을 요구하는** 새 문제를 만들어 반복 학습합니다.

## 사용법
1. 기준이 되는 문제(또는 오답 분석 문서)를 준비한다.
2. 아래 프롬프트에 문제를 넣어 실행한다.
3. 생성된 문제를 풀고, 다시 [풀이 분석 프롬프트](analyze-solution.md)로 점검한다.

---

## 📋 프롬프트 (복사해서 사용)

```text
너는 대한민국 중3 수학 문제를 출제하는 전문가다.
아래 [기준 문제]와 같은 단원·유형·난이도의 **유사 문제 5개**를 만들어줘.

[기준 문제]
(여기에 문제 텍스트 또는 사진 설명을 붙여넣기)

[학생이 약한 부분]
(예: 근의 공식에서 부호 실수 / 판별식 조건 누락)

출제 규칙:
- 같은 개념을 묻되 숫자·상황을 바꿔 그대로 베끼지 않기
- 난이도를 3단계로: 쉬움 2 / 비슷함 2 / 약간 어려움 1
- 학생이 약한 부분을 반드시 한 번씩 건드리는 문제 포함
- 함정/실수 유발 포인트를 의도적으로 1~2개 배치

아래 형식으로 출력:

## 유사 문제 세트 (기준 유형: ____)

### 문제 1 [쉬움]
(문제)

### 문제 2 [쉬움]
(문제)

### 문제 3 [비슷함]
(문제)

### 문제 4 [비슷함]
(문제)

### 문제 5 [약간 어려움]
(문제)
```

## 💡 팁
- 생성된 문제는 `daily/`에 새 기록으로 풀거나, 해당 `mistakes/` 문서의 "반복 방지" 칸에 링크합니다.
- 1주일 뒤 같은 유형을 다시 생성해 풀면 **장기 기억** 정착에 좋습니다.
