# 🤖 prompts/generate-similar-problems.md — 유사 문제 생성

틀린 문제나 약한 패턴과 **같은 사고력을 요구하는** 새 문제를 만들어 반복 학습합니다.

## 사용법
1. 기준이 되는 [문제 페이지](../wiki/math/problems/README.md) 또는 [오답 패턴](../wiki/math/mistakes/README.md)을 고른다.
2. 아래 프롬프트로 유사 문제를 생성한다.
3. 풀고 나면 새 원본으로 `sources/`에 넣어 [compile](compile.md)로 다시 분석한다.

---

## 📋 프롬프트 (복사해서 사용)

```text
너는 대한민국 중3 수학 문제를 출제하는 전문가다.
아래 [기준]과 같은 단원·유형·난이도의 유사 문제 5개를 만들어줘.

[기준 문제/패턴]
(문제 텍스트 또는 wiki/{과목}/mistakes/{패턴}.md 내용 붙여넣기)

[학생이 약한 부분]
(예: 근의 공식 부호 실수 / 판별식 조건 누락 — 오답 패턴 페이지에서)

출제 규칙:
- 같은 개념을 묻되 숫자·상황을 바꿔 그대로 베끼지 않기
- 난이도: 쉬움 2 / 비슷함 2 / 약간 어려움 1
- 학생이 약한 부분을 반드시 건드리는 문제 포함
- 함정/실수 유발 포인트 1~2개 배치

출력:
## 유사 문제 세트 (기준 유형: ____)
### 문제 1 [쉬움]
...
### 문제 5 [약간 어려움]
...

<details><summary>정답·풀이</summary>
1. 답 / 핵심:
...
</details>
```

---

## 💡 팁
- 푼 유사 문제도 `sources/`에 넣고 compile하면 약점 호전이 데이터로 추적됩니다.
- 1주 뒤 같은 패턴을 다시 생성해 풀면 장기 기억에 좋습니다.
- 해당 [오답 패턴 페이지](../wiki/math/mistakes/README.md)의 "호전 추적"을 갱신하세요.
