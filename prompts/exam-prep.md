# 🤖 프롬프트 — 시험 대비 자료 생성 (Exam Prep)

누적된 오답·약점 데이터를 **시험 범위 기준 대비 자료**로 재구성합니다.

## 사용법
1. 시험 범위를 확정한다.
2. 범위에 해당하는 `weakness-map.md`, `mistakes/` 내용을 모은다.
3. 아래 프롬프트로 대비 계획 + 예상 문제를 받는다.
4. 결과를 [`exams/`](../content/exams/README.md)의 시험 문서에 정리한다.
