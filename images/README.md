# 🖼️ 이미지 저장소 (Images)

문제 사진과 풀이 사진을 보관하는 곳입니다.

---

## 📁 폴더 구조 (연/월)

```text
images/
└─ YYYY/
   └─ MM/
      ├─ YYYY-MM-DD-q1.jpg       # 문제 사진
      ├─ YYYY-MM-DD-q1-sol.jpg   # 풀이 사진
      └─ YYYY-MM-DD-q2.jpg
```

예) `images/2026/06/2026-06-07-q1.jpg`

---

## 🏷️ 파일 이름 규칙

`YYYY-MM-DD-q{번호}[-sol].{확장자}`

| 부분 | 의미 |
|------|------|
| `YYYY-MM-DD` | 푼 날짜 |
| `q1`, `q2` | 그날의 문제 번호 |
| `-sol` | 풀이 사진임을 표시 (없으면 문제 사진) |

---

## 🔗 Markdown에서 삽입하기

`content/daily/`에서는 상대경로로:
```markdown
![문제1](../../images/2026/06/2026-06-07-q1.jpg)
![풀이1](../../images/2026/06/2026-06-07-q1-sol.jpg)
```

---

## ⚠️ 주의사항

- 📸 **개인정보 가리기:** 사진에 이름·학교·이름표가 보이면 가린 뒤 올린다. ([privacy](../docs/privacy.md))
- 📦 **용량 관리:** 휴대폰 원본은 큽니다. 가로 1600px 정도로 줄이면 충분합니다.
- 🔁 **파일명 일관성:** 위 규칙을 지켜야 나중에 찾기 쉽습니다.
