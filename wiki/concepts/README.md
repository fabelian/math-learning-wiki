# 📘 concepts/ — 개념 엔티티 페이지

단원/개념을 **누적되는 엔티티**로 관리합니다. 새 문제가 컴파일될 때마다 LLM이 해당 개념 페이지를 갱신합니다.

---

## 📌 규칙

- 파일명: `{단원}.md` (예: `이차방정식.md`)
- 형식: [`../templates/concept-note-template.md`](../templates/concept-note-template.md)
- 새 사실이 들어오면 **덮어쓰지 말고 통합**하고, 기존 서술과 모순되면 명시한다 ([CLAUDE.md §4.2](../../CLAUDE.md)).
- "자주 하는 실수" 표는 컴파일된 [문제](../problems/README.md)·[오답 패턴](../mistakes/README.md)과 링크로 연결.
