# 📘 concepts/ — 개념 엔티티 페이지

단원/개념을 **누적되는 엔티티**로 관리합니다. 새 문제가 컴파일될 때마다 LLM이 해당 개념 페이지를 갱신합니다.

> 참고: 실제 학습 내용은 **고등 미적분(수II) 선행** 수준입니다(2026-06-07 진단 기준). 아래 고등 단원을 주로 사용합니다.

## 📌 규칙
