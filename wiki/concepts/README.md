## 🏫 중3 학교 진도 (별도 트랙)

| 단원 | 상태 |
|------|------|
| 제곱근과 실수 / 인수분해 / 이차방정식 / 이차함수 | 미작성 |
| 삼각비 / 원의 성질 / 통계 / 피타고라스 정리 | 미작성 |
