# 🧭 고교 선택 — 강남권 vs 외대부고

> 강사 설명 기반. 정책·진학 수치는 연도별로 다르니 공식자료 확인.

## 🤔 왜 고민이 커졌나
- **정시 내신 반영** 흐름 → 강남권 일반고 잔류 vs 외대부고 진학 고민 증가.

## ✅ 판단 기준 (강사 의견)
- **영어 실력이 대원외고·한영외고 학생보다 낫다 → 외대부고 고려.** 아니면 강남권 권유.
- "수학·과학만 잘해서 외대부고"는 **오판**. **객관 데이터(TEPS 문법 점수 등)**로 냉정히 판단.
- 외대부고엔 유학파·전국 극상위가 모임 → "수발 잘함"만으론 오산.

## 🏫 학교별 영어시험 성향 (강사 비교)
| 학교 | 성향 |
|------|------|
| **외대부고** | **표준화·100% 객관식**(SAT 전통), 이상한 문제 적음, 측정 정확 |
| **하나고** | 지필 + **라이팅(논술)** 포함 → 만점 어려움, 상대적으로 더 까다로움 |
| **한영외고** | 문법 없음, "문제를 위한 문제" 성향 |
| **대원외고** | 서술형 포함 |
| **강남 일반고**(예: 숙명여고 등) | 텍스트는 쉬우나(교과서·모의고사 수준) **변별 위해 문제를 꼬아** 까다로움 |

## 🏠 기숙형·융합형 관점
- 기숙형(외대부고)은 24시간 공동생활 → **융합형(STEAM)·협업 교육에 유리**.
- 창체(자율동아리·봉사·진로)가 미국 입학사정관제의 EC/character에 해당. (→ 아래 **AI시대 입시 흐름**)

## 🩺 진학 실적 — 숫자 읽는 법 (강사 언급, 검증 필요)
- "상산고가 의대 더 많이 간다"는 말의 함정: **모집 규모가 다름**.
- 예시(강사): 외대부고 자연계 ≈ 140명 중 의대 ~60명, 상산고 ≈ 450명 중 ~90~100명 → **비율로 보면 외대부고가 높음**.
- ⚠️ 수치는 연도·집계 기준에 따라 다름. **공식 진학자료로 확인.**

## 🌐 AI시대 입시 흐름 (참고)
- **STEM → STEAM**(A=인문·사회·예술·철학). 수능 약화·수시 강세·대학 자율성↑·**면접 강화**(서울대 MMI 등).
- 고교학점제: 1학년 공통 → 2학년 선택(전공적합성)에서 **"깊고 도전적인 과목" 선택** 권장.
- 우리 입시 ≈ 미국 입학사정관제 + IB 요소 융합.
