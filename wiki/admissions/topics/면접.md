# 🎤 면접 대비

> 자기주도학습전형 2단계 핵심. 강사 설명 기반 — 형식·시간은 연도별 변동, 공식 안내 확인.

## 🧠 면접의 성격
- **합격시키는 시험이 아니라, "정말 본인이 했는지" 검증·비판하는 시험.**
- 질문은 **자소서·생기부에서** 나온다. 근거가 생기부에 없으면(AI·외부 도움 의심) **난이도↑**.
- **면접 난이도 = 생기부·자소서 준비도** (→ [학교 프로필](../schools/용인외대부고.md)).

## ⏱️ 형식 (외대부고 — 강사 언급)
- 최근: **약 16분, 2문제.**
- **2문제형**: 문제당 답변 ~5분 + **꼬리질문 ~2분 30초**.
- **3문제형**(A·B·C): 문제당 답변 ~5분 + 질문 ~4분.
- 답변 분량을 **5분 / 2분 30초 / 1분 30초** 버전으로 준비해두면 좋다(시작·전개·마무리 구조).

## 🔁 꼬리질문(압박) 7유형의 뼈대
- "**이걸 했다는데, 반대 입장/반대 상황은 어떻게?**"
- "**그 숨은 이유·동기는 무엇?**"
- → 그래서 자소서 소재는 **내가 깊이 아는 것**이어야 어떤 꼬리질문도 받아친다. ([자소서 황금률](자기소개서.md))

## 🧩 정답 없는 문제(논술형 면접)
- 바칼로레아式·AI 윤리式 질문: "기술이 철학자/과학자에게 어떤 도움을 주는가", "AGI에 앞서 필요한 윤리 기준을 인간 존엄·생명권 관점에서 말하라" 등.
- **정답이 없다 → 본인 생각을 논리적 근거 + 배경지식으로** 전개하면 된다.
- **배경지식 = 직접경험(창체·동아리) + 간접경험(독서·자료조사)** 의 결합.

## 🌟 학교가 원하는 인재상
- **"함께 하고 싶은 사람"**: 협업 잘하고, 같이 더 하고 싶게 만드는 사람.
- 맥락·**스토리텔링** 능력. 정답 암기형이 아니라 사고·논리·인간성.

## ✅ 대비 체크
- 자소서 모든 문장에 "왜?/반대는?/그래서?" 자문하며 꼬리질문 예행연습.
- 답변 길이별(5/2.5/1.5분) 템플릿 연습.
- 시사·윤리 이슈 배경지식 + 본인 논리 정리.
