# 🧩 문제 2026-06-07-q7  👏 까다로운 추론 정답

> 단원: **사차함수 그래프 추론** · 결과: **정답 ✅**

## 🔗 원본
- [시작테스트 q7](../../../sources/math/2026/06/2026-06-07-시작테스트.md#q7)

## 🧠 정답 분석
- 최고차 음수 + $f'=0$ 세 실근 → $x=\alpha$ 극대, $x=\beta$ 극소, $x=\gamma$ 극대. ($f(\beta)<f(\alpha),\ f(\beta)<f(\gamma)$)
- **ㄱ** $x=\beta$ 극솟값 → **참**.
- **ㄴ** $f(\alpha)f(\beta)f(\gamma)>0$ 인 모든 경우를 따져보면 $f(x)=0$ 은 항상 서로 다른 **두 실근** → **참**.
- **ㄷ** $f(\alpha)<0$ 이면 (이때 $f(\beta)<0,\ f(\gamma)>0$) 두 실근이 모두 $\beta$보다 **크다** → "$\beta$보다 작은 실근" **거짓**.
- 정답 **③ ㄱ,ㄴ**.

## ✍️ 학생 답
- ③ ㄱ,ㄴ → **정답.** (④에도 표시 흔적 있으나 최종 ③로 판단)

## 👍 장점 (칭찬)
- 음수 최고차 사차함수의 증감(극대-극소-극대)을 정확히 세움. 부호 조건으로 근의 개수를 경우 나눠 추론하는 고난도 사고를 해냄.

## 🔍 점검 포인트
- ㄷ에서 "$f(\alpha)<0$ ⇒ $f(\gamma)>0$" 가 강제되는 이유(곱 부호)를 한 줄로 설명할 수 있는지 확인.
