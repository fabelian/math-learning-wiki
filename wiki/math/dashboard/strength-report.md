## 2. 반복 숙달한 핵심 기법 (강점의 실체)

### ① 미분계수의 정의로 극한 변환 ⭐⭐⭐
복잡한 극한을 $\lim\frac{f(x)-f(a)}{x-a}=f'(a)$ 로 꿰뚫어 봄.
근거: [C55](../problems/2026-06-08-c55.md), [C69](../problems/2026-06-08-c69.md), [C141](../problems/2026-06-08-c141.md), [2차 q8](../problems/2026-06-07-t2-q8.md), [2차 q9](../problems/2026-06-07-t2-q9.md), [A159](../problems/2026-06-08-a159.md)

### ② 다항식 미정계수 결정 ⭐⭐⭐ (최빈·고난도)
"$f(a)=0 ⇒ (x-a)$ 인수", "두 함수가 공통점/직선 공유 ⇒ 공통 인수", "유한극한 ⇒ 분자가 분모의 0 상쇄".
근거: [진도 q10](../problems/2026-06-08-pt-q10.md), [진도 q11](../problems/2026-06-08-pt-q11.md), [A176](../problems/2026-06-08-a176.md), [B120](../problems/2026-06-08-b120.md), [B140](../problems/2026-06-08-b140.md), [C175](../problems/2026-06-08-c175.md), [2차 q11](../problems/2026-06-07-t2-q11.md)

### ③ 미분가능성·연속 조건으로 함수 확정 ⭐⭐⭐
절댓값 안이 중근(꺾임 없음), 분수꼴 분자가 분모의 0 상쇄, 우함수 ⇒ 도함수는 기함수.
근거: [2차 q12](../problems/2026-06-07-t2-q12.md), [C119](../problems/2026-06-08-c119.md), [C175](../problems/2026-06-08-c175.md), [B83](../problems/2026-06-08-b83.md), [단원 q12](../problems/2026-06-07-ut-q12.md), [C98](../problems/2026-06-08-c98.md)

### ④ 근과 계수의 관계·대칭식 ⭐⭐⭐
합·곱으로 대칭식 계산, 부호/절댓값 처리, "근 대입으로 차수 낮추기".
근거: [진도 q3](../problems/2026-06-08-pt-q3.md), [진도 q6](../problems/2026-06-08-pt-q6.md), [진도 q7](../problems/2026-06-08-pt-q7.md), [진도 q8](../problems/2026-06-08-pt-q8.md), [진도 q12](../problems/2026-06-08-pt-q12.md)

### ⑤ 극한 계산 기본기 ⭐⭐
$\infty/\infty$ 최고차 비교, $0/0$ 유리화, 샌드위치 정리.
근거: [2차 q3~q6](../problems/2026-06-07-t2-q3.md), [단원 q4~q6](../problems/2026-06-07-ut-q4.md)

근거: [1차 q9](../problems/2026-06-07-q9.md), [단원 q10](../problems/2026-06-07-ut-q10.md), [1차 q7](../problems/2026-06-07-q7.md), [A151](../problems/2026-06-08-a151.md), [B116](../problems/2026-06-08-b116.md)

## 3. 유일한 주의점 (보강 1개)

- **음수의 제곱근 부호**: $a<0,\ b<0$ 이면 $\sqrt a\sqrt b=-\sqrt{ab}$ (그냥 $\sqrt{ab}$ 아님).
  - 발생: [진도 q4](../problems/2026-06-08-pt-q4.md) — 처음 $-2\sqrt2$ → $2\sqrt2$ 로 **스스로 정정**.
  - 처방: 켤레/허근·음수 제곱근이 나오면 부호 한 번 더 확인. (반복 시 약점 등록 예정, 현재는 1회·정정됨)

## 4. 종합 평가 & 추천

**평가:** 미적분Ⅰ(극한·미분·미분가능성)과 이차방정식의 **개념 이해 + 고난도 응용**이 모두 탄탄. 특히 *다항식 결정형*과 *미분가능 조건형* 같은 상위권 변별 문제를 안정적으로 해결.

**추천 다음 단계:**
1. **적분(정적분·넓이)** 으로 진도 확장 — 미분 기초가 탄탄해 무리 없음.
2. **잠정 9문항 확정** — 그래프/원문만 선명히 받으면 값 확정 ([약점 지도](weakness-map.md) 참고).
3. **실전 속도 훈련** — 정확도는 충분하니, 고난도 유형 풀이 시간 단축에 집중.
4. **부호 점검 루틴** — 음수 제곱근/이항 부호만 가볍게 셀프체크.

> 다음 시험 대비 시 §2의 ①~④ 기법 문제를 모아 빠르게 복습하면 효율적입니다.
