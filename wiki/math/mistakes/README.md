# ❌ mistakes/ — 오답 패턴 엔티티 페이지

**반복되는 실수 유형 하나 = 엔티티 페이지 하나.** (예: `부호처리실수.md`)
같은 패턴이 여러 문제에서 나오면, 한 페이지에 모아 추적합니다.

> ⚠️ 개별 오답 분석은 [문제 페이지](../problems/README.md)에 있습니다.
> 여기는 그 오답들을 **유형별로 묶은 상위 패턴**입니다.

---

## 📌 규칙

- 파일명: `{패턴}.md` (예: `부호처리실수.md`, `판별식조건누락.md`)
- 형식: [`../templates/mistake-pattern-template.md`](../../templates/mistake-pattern-template.md)
- 새 발생 시 `occurrences`에 문제 링크 추가, `repeat_count` +1.
- 각 패턴은 [오답 원인 태그](#오답-원인-태그) 중 하나에 속한다.

---

## 🏷️ 오답 원인 태그 (고정 어휘)

[CLAUDE.md §6](https://github.com/fabelian/math-learning-wiki/blob/main/CLAUDE.md)과 동일. compile 시 반드시 이 태그만 사용.

| 태그 | 의미 |
|------|------|
| `#계산실수` | 개념은 알지만 연산·부호·이항 실수 |
| `#개념부족` | 정의·공식·정리를 몰라서 |
| `#문제해석오류` | 조건을 놓치거나 잘못 읽음 |
| `#풀이전략오류` | 접근 방법 자체가 오류/비효율 |
| `#시간부족` | 방법은 알지만 시간 내 못 풂 |
| `#실수누락` | 검산·단위·범위 확인 안 함 |
| `#서술미흡` | 답은 맞지만 풀이 서술 부족 |
